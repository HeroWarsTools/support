Prepare your Firefox to open Multi Accounts

1. Download Firefox Multi-Account Containers

Official extension to manage containers and separate your online identities.

Get Add-on

2. Download Open External Links In a Container

Allows you to force external links to open in specific containers.

Get Add-on

3. Download user.js Configuration

Open the link below and press Ctrl+S to save the file to your computer.

Download user.js

4. Open Profile Folder

Copy and paste about:support into your address bar (or click the button below), then find the "Profile Folder" entry and click "Open Folder" (or "Show in Finder").

Open about:support

Note: If the button above doesn't open, please manually type about:support in your URL bar.

5. Close Firefox

Make sure Firefox is completely closed before proceeding to the next step.

6. Install user.js

Paste the user.js file you downloaded into the Profile Folder you opened in Step 4.

7. Re-open Firefox

Start Firefox again to apply all the new settings.
